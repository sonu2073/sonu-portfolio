## 🧾 Overview

A fully responsive, single-file personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no build tools, no dependencies. Designed with a dark, futuristic aesthetic featuring smooth animations, glowing accents, and a custom cursor experience.

> Built to showcase projects, skills, and experience in a way that's memorable and professional.

## ✨ Features

- **Single File** — Entire portfolio in one `index.html` file, zero dependencies
- **Custom Cursor** — Glowing dot with trailing ring that reacts to hover states
- **Scroll Reveal Animations** — Elements animate into view as you scroll using IntersectionObserver
- **Hero Entrance Animations** — Staggered fade-up sequence on page load
- **Mouse-tracking Glow** — Project cards follow your cursor with a radial light effect
- **Floating Orbs** — Ambient animated background orbs for depth and atmosphere
- **Noise Texture Overlay** — Subtle grain for a premium, tactile feel
- **Active Nav Highlight** — Navigation links highlight based on current scroll section
- **Fully Responsive** — Works cleanly on mobile, tablet, and desktop
- **Smooth Scrolling** — Native smooth scroll between all sections

## 🧰 Tech Used

This portfolio uses **zero external dependencies** — everything is built with:

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, grid, flexbox, backdrop-filter
- **Vanilla JavaScript** — IntersectionObserver, mousemove tracking, scroll events
- **Google Fonts** — `Syne` (display) + `DM Sans` (body)

## 🏁 Getting Started

No installation needed. Just open the file:

```bash
# Clone the repository
git clone https://github.com/SonuMeena/portfolio.git

# Navigate into the folder
cd portfolio

# Open directly in your browser
open index.html
```

Or simply **double-click** `index.html` — it works instantly in any modern browser.

---

## ☁️ Deployment

### Option 1 — Netlify Drop *(Fastest — 10 seconds)*
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop `index.html` onto the page
3. Get a live URL instantly ✅

### Option 2 — GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://sonumeena.github.io/portfolio` ✅

### Option 3 — Vercel
```bash
npm install -g vercel
vercel
```
Follow the prompts — live in under a minute ✅
